# 🎬 GemiFlix

### Netflix-Inspired UI. Gemini-Powered Intelligence.

GemiFlix is a modern movie discovery application inspired by Netflix. It provides a beautiful streaming experience with AI-powered movie recommendations, real-time movie data, and a clean, responsive interface.

---

## ✨ Features

- 🎥 Browse popular, top-rated, trending, and upcoming movies
- 🤖 AI-powered movie recommendations
- 🔍 Smart movie search
- 🎬 Watch movie trailers
- ❤️ Save movies to My List
- 🔐 Secure authentication
- 📱 Fully responsive design
- ✨ Smooth animations and modern UI

---

## 🛠️ Built With

- React.js
- Redux Toolkit
- Tailwind CSS
- Firebase Authentication
- TMDB API
- Google Gemini AI
